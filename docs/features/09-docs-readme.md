# F9 — Финал: README и проверка запуска

**Ветка:** `feature/docs-readme`

## Scope
Завершающая фича: документация и проверка работоспособности.

- **README**: описание проекта, архитектура (кратко), стек, структура monorepo,
  инструкции запуска (`npm install`, `npm run dev`), переменные окружения (таблица),
  команды (`db:migrate`, `db:seed`, `test`, `build`, `lint`, e2e), реализованные
  возможности, известные ограничения, дальнейшее развитие (accompaniment engine, MIDI export).
- Обновить `docs/` под фактическую реализацию (если были отклонения — отражены в ADR).
- `.env.example` — полный и актуальный.
- Финальный прогон **всех** тестов (unit + integration + e2e).
- Проверка чистого сценария: clone → `npm install` → `npm run db:migrate && npm run db:seed`
  → `npm run dev` → web и api поднимаются, dev-login работает, метроном играет по сетке.

## Зависимости
F0–F8 (все предыдущие).

## Тесты
- Полный прогон test-сьюта всех workspaces + Playwright.

## Definition of Done
- Проект запускается из чистого клона по инструкциям README.
- Выведены deliverables: архитектура, возможности, ограничения, запуск, env vars, что улучшить.
