# ALL_CHORDS — Справочник ступеней DSL

> **Назначение:** Полный перечень ступеней, поддерживаемых в DSL-сетке упражнений с аккордами (источник «Сетка (ступени)»).
> **Аудитория:** Пользователи тренажёра, разработчики, AI-агенты.
> **Дата:** 2026-06-16

---

## 1. Концепция

В поле «Сетка (ступени)» прогрессия записывается **ступенями**, а не абсолютными аккордами. Ступень не зависит от тональности — это положение аккорда относительно тоники. Генератор раскрывает каждую ступень в конкретный аккорд для каждой выбранной тональности.

Пример: `| IIm7 | V7 | Imaj7 |`

| Тональность | Раскрытие           |
| ----------- | ------------------- |
| C           | `Dm7 → G7 → Cmaj7`  |
| F           | `Gm7 → C7 → Fmaj7`  |
| Bb          | `Cm7 → F7 → Bbmaj7` |

**Синтаксис записи:**

- Такты разделяются вертикальной чертой `|`.
- Несколько ступеней в одном такте разделяются пробелами или запятыми: `| IIm7 V7 | Imaj7 |`.
- Перевод строки приравнивается к пробелу (можно писать в несколько строк).

**Формула ступени:**

```
[♭/♯] <римская цифра> [суффикс качества] [/ <ступень-цель>]
   │         │                │                   │
альтерация  I–VII        m7, maj7, 7…      вторичная доминанта
```

> ⚠️ **Регистр римской цифры не важен** — `ii` и `II` дают одну и ту же ступень. Качество аккорда задаётся **только суффиксом**, а не регистром. Без суффикса ступень трактуется как мажорное трезвучие.

## 2. Диатонические ступени (мажор)

Семь основных ступеней мажорной тональности.

| Ступень | Полутонов от тоники (degree) | Пример в C | Типичная функция |
| ------- | :--------------------------: | ---------- | ---------------- |
| `I`     |              0               | C          | Тоника           |
| `II`    |              2               | D          | Супертоника      |
| `III`   |              4               | E          | Медианта         |
| `IV`    |              5               | F          | Субдоминанта     |
| `V`     |              7               | G          | Доминанта        |
| `VI`    |              9               | A          | Субмедианта      |
| `VII`   |              11              | B          | Вводный тон      |

---

## 3. Хроматические (альтерированные) ступени

Префикс `♭`/`b` понижает ступень на полутон, `♯`/`#` — повышает.

| Ступень | degree | Пример в C | Частое применение                                        |
| ------- | :----: | ---------- | -------------------------------------------------------- |
| `♭II`   |   1    | Db (C#)    | Неаполитанская / тритоновая замена доминанты (`♭IImaj7`) |
| `♯I`    |   1    | C#         | Хроматический проходящий аккорд                          |
| `♭III`  |   3    | Eb         | Заимствование из параллельного минора                    |
| `♯IV`   |   6    | F#         | `♯IVm7b5` — подход к V, линия баса                       |
| `♭V`    |   6    | Gb         | Тритоновая замена                                        |
| `♭VI`   |   8    | Ab         | Заимствование из минора (`♭VImaj7`)                      |
| `♭VII`  |   10   | Bb         | Бэкдор-доминанта (`♭VII7`), миксолидийский колорит       |

> Запись `♭` и `b` эквивалентны (`♭II` = `bII`), как и `♯` и `#` (`♯IV` = `#IV`). В превью альтерация всегда отображается музыкальными знаками `♭`/`♯`.

## 4. Суффиксы качества

Суффикс добавляется сразу после римской цифры. Используется тот же словарь качеств, что и для обычных аккордов.

| Суффикс | Качество                    | Пример ступени | В тональности C |
| ------- | --------------------------- | -------------- | --------------- |
| _(нет)_ | Мажорное трезвучие          | `I`            | C               |
| `maj7`  | Большой мажорный септаккорд | `Imaj7`        | Cmaj7           |
| `maj9`  | Мажорный нонаккорд          | `Imaj9`        | Cmaj9           |
| `6`     | Мажорный секстаккорд        | `I6`           | C6              |
| `m`     | Минорное трезвучие          | `IIm`          | Dm              |
| `m7`    | Малый минорный септаккорд   | `IIm7`         | Dm7             |
| `m6`    | Минорный секстаккорд        | `IVm6`         | Fm6             |
| `m9`    | Минорный нонаккорд          | `IIm9`         | Dm9             |
| `7`     | Доминантсептаккорд          | `V7`           | G7              |
| `9`     | Доминантнонаккорд           | `V9`           | G9              |
| `11`    | Ундецимаккорд               | `V11`          | G11             |
| `13`    | Терцдецимаккорд             | `V13`          | G13             |
| `m7b5`  | Полууменьшённый (ø)         | `VIIm7b5`      | Bm7b5           |
| `dim`   | Уменьшённое трезвучие       | `VIIdim`       | Bdim            |
| `dim7`  | Уменьшённый септаккорд      | `VIIdim7`      | Bdim7           |
| `aug`   | Увеличенное трезвучие       | `Vaug`         | Gaug            |
| `sus4`  | Задержание кварты           | `Vsus4`        | Gsus4           |
| `sus2`  | Задержание секунды          | `Vsus2`        | Gsus2           |
| `5`     | Квинтаккорд (power)         | `I5`           | C5              |

**Альтерации и расширения** комбинируются с доминантами:

| Суффикс | Пример  | В C   | Назначение                          |
| ------- | ------- | ----- | ----------------------------------- |
| `7b9`   | `V7b9`  | G7b9  | Минорная каденция, разрешение в i   |
| `7#9`   | `V7#9`  | G7#9  | «Hendrix», блюзовый колорит         |
| `7b5`   | `V7b5`  | G7b5  | Альтерированная доминанта           |
| `7#5`   | `V7#5`  | G7#5  | Увеличенная доминанта               |
| `7#11`  | `V7#11` | G7#11 | Лидийская доминанта                 |
| `7b13`  | `V7b13` | G7b13 | Альтерированная доминанта           |
| `7alt`  | `V7alt` | G7alt | Полностью альтерированная доминанта |

## 5. Вторичные доминанты и относительные ступени

Запись `X/Y` означает «ступень `X`, построенная относительно ступени `Y` как временной тоники». Степень вычисляется как сумма: `degree = (degree(X) + degree(Y)) % 12`, качество берётся от левой части `X`.

| Запись    | Расшифровка                  |    degree    | В тональности C |
| --------- | ---------------------------- | :----------: | --------------- |
| `V7/V`    | Доминанта к доминанте        | (7+7)%12 = 2 | D7              |
| `V7/ii`   | Доминанта ко второй ступени  | (7+2)%12 = 9 | A7              |
| `V7/IV`   | Доминанта к субдоминанте     | (7+5)%12 = 0 | C7              |
| `V7/vi`   | Доминанта к шестой ступени   | (7+9)%12 = 4 | E7              |
| `IIm7/IV` | Минорная вторая ступень к IV | (2+5)%12 = 7 | Gm7             |

> Поддерживается **один** уровень вложенности (`X/Y`, но не `X/Y/Z`).

## 6. Примеры готовых прогрессий

```
| IIm7 | V7 | Imaj7 |              ← ii–V–I (мажор)
| IIm7b5 | V7b9 | Im7 |            ← ii–V–i (минор)
| I6 | VIm7 | IIm7 | V7 |          ← джазовый turnaround
| Imaj7 | VI7 | IIm7 | V7 |        ← rhythm changes (фрагмент A)
| ♭IImaj7 | Imaj7 |                ← тритоновая замена
| V7/V | V7 | Imaj7 |              ← вторичная доминанта
```

## 7. Ошибки валидации

Поле подсвечивается красным, если ступень не распознана:

- **Неверная ступень** — токен не начинается с римской цифры I–VII (например, абсолютный аккорд `Dm7` или `Xyz`).
- **Неизвестное качество аккорда** — суффикс не входит в словарь качеств (например, `IIzzz`).
- **Неверная цель вторичной доминанты** — правая часть `X/Y` не является ступенью (например, `V7/Q`).

Ошибки собираются по всем тактам сразу и показываются с указанием проблемного токена.
